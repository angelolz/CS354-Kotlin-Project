What is Kotlin?

Kotlin is a language developed for the Java Virtual Machine by IntelliJ, the makers of IDEA. Named for an island in the Baltic sea, it aims to be better than Java! Google announced in 2019 that Kotlin would be the preferred language for Android application development. Although Kotlin typically complies to Java bytecode, it is also commonly compiled to JavaScript and machine code (for mobile processors).

Kotlin is an open-source language that features statically-typed variables (i.e., you don’t need to explicitly declare them). Variables can be declared read only or mutable. In Kotlin, semicolons are optional, at least most of the time… And, of course, Kotlin is object-oriented and prominently features the functional paradigm.

Online Interactive Shell

Kotlin provides an interactive shell where you can write Kotlin code and run it all in your browser. This is great for learning the basic syntax of Kotlin.

Here are a couple of IDEs for writing Kotlin code:

These are all great IDEs, but the most preferred IDE to use with Kotlin is IntelliJ IDEA. This is because it has "first-class support" for Kotlin, as well as various project templates, code completion, and refactoring/debugging tools. If you're familiar with Java's debugger and refactoring tools, that is sufficient enough experience to use in Kotlin. Besides, IntelliJ IDEA is made by the same people that made Kotlin!

Compiling Kotlin Code

The general way to compile your Kotlin code using IntelliJ IDEA and the Gradle build tool. IDEA already has Kotlin support without needing extra components. If you desire to compile your Kotlin code through the command-line, you can download a Kotlin compiler that's native to your operating system in Kotlin's releases page.

You can see the usage of the native compiler in our code examples below.

Sum and Decrease

Written by: Angel

This code showcases loops (for/while loops) and user input in Kotlin.

fun main(args: Array<String>) {
    println("Program arguments: ${args.joinToString()}")

    try {
        var sum = 0;
        //for loop here
        for(arg: String in args) {
            sum += arg.toInt();
        }

        println("The total sum is: $sum")
        println("Decreasing sum...")

        //while loop here
        while(sum > 0) {
            sum--
            println(sum)
        }
    }

    catch(e: NumberFormatException) {
        println("One or more of your arguments isn't a number.")
    }
}

Bank Program

Written by: Landon

This code showcases object-oriented programming in Kotlin.

//Account.kt
abstract class Account(number: Int, cus: Customer, balance:Double) {
    var accountNumber = 0
    var customer: Customer? = null
    var accountBalance = 0.0

    abstract fun accrue(rate: Double)

    fun deposit(amount: Int) {
        accountBalance+=amount
    }

    fun withdraw(amount:Int) {
        accountBalance-=amount
    }

    override fun toString(): String = accountNumber.toString() +':'+customer+':'+ accountBalance
}

//CheckingAccount.kt
class CheckingAccount(number: Int, cus: Customer, balance:Double): Account(number,cus,balance) {
    init {
        customer = cus
        accountNumber = number
        accountBalance = balance
    }

    override fun accrue(rate:Double) {}
}

//SavingAccount.kt
class SavingsAccount(number: Int, cus: Customer, balance:Double): Account(number,cus,balance) {
    var interest = 0.0

    init {
        customer = cus
        accountNumber = number
        accountBalance = balance
        interest = 0.0
    }

    override fun accrue(rate: Double) {
        interest+= accountBalance*rate
		accountBalance+=accountBalance*rate
    }
}

//Customer.kt
class Customer(CustomerName: String) {
    var c = ""
    init {
        c=CustomerName
    }

    override fun toString(): String = c
}

//Bank.kt (main class!)
class Bank() {
    var hash = HashSet<Account>()

    fun add(acc: Account) {
        hash.add(acc)
    }

    fun accrue(newrate: Double){
        for (i in hash) {
            i.accrue(newrate)
        }
    }

    override fun toString(): String{
        var returnString = ""
        for (i in hash) {
            returnString+= i.toString()+'\n'
        }
        return returnString
    }
}

fun main() {
    var bank = Bank()
    var c = Customer("Ann")
    bank.add(CheckingAccount(1001,c,100.00))
    bank.add(SavingsAccount(1002,c,200.00))
    bank.accrue(0.02)
    println(bank.toString())
}
